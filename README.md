Squall Reverb for the Korg NTS-1 mkii,
can use as REVERB EFFECTS.

AS of the latest commit, this effect is not functioning yet.

Parameters
shape (NTS-1 mkII: A)
TIME.
alt (NTS-1 mkII: B)
DEPTH.
shift depth
dry/wet mixing ratio.

How to build
Clone (or download) this project.
Copy to under platform/nts-1_mkii.
See README.md under platform/nts-1_mkii.
